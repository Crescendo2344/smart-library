Steps how to run my file:

1. Copy the smart-library folder on the repository.
2. Paste it to htdocs folder on xampp folder located on windows:c.
3. Make sure your Apache and MySQL on xampp server is running.
4. Open on browser "localhost/phpmyadmin/" and create a database named "smart_library".
5. After you enter the database name click import on the top and choose the file smart_library.sql to import the database.
6. Open the file on browser type "localhost/smart-library/landing.php".
7. Head to the users table to find the users registered (note that the users in the table is example user) you can use the email to log in.
